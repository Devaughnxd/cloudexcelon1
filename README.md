# BTP SecuriSCOPE

Production-ready lightweight PHP 8.3 website for securiscope.cloudexcelon.com.

Deploy the repository root directly to the Plesk domain folder. `index.php` must be in the domain root.

Pages: Home, About, Services, News, Contact.

## Quality Process

- `docs/website-audit.md`: running issue log. Log issues before fixing them.
- `docs/competitive-benchmark.md`: visual, UX, and information architecture benchmark guide.
- `docs/content-source-policy.md`: BTP-only factual content policy.
- `docs/qos-checklist.md`: pre-deployment quality checklist.

Competitor sites may be used to judge design quality, service-page structure, and conversion flow. They must not be used as factual sources for BTP claims or service copy.
