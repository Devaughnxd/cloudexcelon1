# BTP Design System

Typography: Segoe UI/Inter-style system font, strong enterprise headlines, readable body copy.

Palette: black `#000000`, white `#FFFFFF`, blue `#2AA8FF`, deep blue `#0067F0`, red `#FF3B30`, plus brand accent tokens.

Buttons: primary red CTA, secondary blue action, outline discovery action.

Cards: 8px radius, light border, subtle shadow, compact enterprise spacing.

Spacing: mobile-first sections using responsive clamp spacing and a 1200px max container.

Icons: clean text-based service markers and consistent BTP bar-logo system.

Forms: one lead form only, with CRM-ready field names: `full_name`, `company_name`, `email`, `phone`, `message`, `brand`.
