# BTP Brand Websites

Separate deploy-root PHP websites generated from one BTP design system.

Each site includes website audit, Plesk preview audit, competitive benchmark, content-source, and QOS documentation under `docs/`.

- `sites/praas`: BTP PraaS (praas.cloudexcelon.com)
- `sites/techadvisors`: BTP TechAdvisors (techadvisors.cloudexcelon.com)
- `sites/securiscope`: BTP SecuriSCOPE (securiscope.cloudexcelon.com)
- `sites/managesp`: BTP ManageSP (managesp.cloudexcelon.com)
- `sites/cloudexcelon`: BTP CloudEXCELON (dev.cloudexcelon.com)
- `sites/codeignite`: BTP CodeIGNITE (codeignite.cloudexcelon.com)
- `sites/datastaisis`: BTP DatastAIsis (datastaisis.cloudexcelon.com)
